# 🚀 SIRAIL - Enterprise Web Dashboard

A modern **enterprise-grade web application** built for SIRAIL Group using **React, TypeScript, and Vite**.  
This project provides a fast, scalable, and responsive interface for managing internal company operations.

---

## 🧠 Project Overview

SIRAIL is a professional dashboard system designed to improve internal workflow and user experience inside the company.

It focuses on:
- Performance ⚡
- Scalability 📈
- Clean UI/UX 🎨
- Maintainable code 🧩

---

## ✨ Features

- 🖥️ Modern dashboard interface
- 🔐 Authentication UI (login system)
- 📊 Data visualization & management
- 📱 Fully responsive design
- ⚡ Fast build with Vite
- 🧩 Component-based architecture

---

## 🛠️ Tech Stack

- React.js
- TypeScript
- Vite
- pnpm
- HTML5 / CSS3

## 🚀 Installation & Setup

git clone <your-repo-url>
cd sirail-source

pnpm install

pnpm dev

---

## 🏗️ Build for Production

pnpm build

pnpm preview

---

## ⚙️ Environment Variables

Create a `.env` file:

VITE_API_URL=http://localhost:3000
